# Dm everyone Bot

## Requirements

-   `Node.js` (v18+ or higher recommended)
-   `discord.js` (v14)

## Installation

1. **Download or fork the repo**:

2. **Installing the dependencies**:

    ```bash
    npm install
    ```

3. **Configuration**:

   `config.json` - 

    ```json
    {
        "PREFIX": "YOUR_PREFIX"
    }
    ```
Create a `.env` file and paste the token like this -
```
TOKEN = token here
```
